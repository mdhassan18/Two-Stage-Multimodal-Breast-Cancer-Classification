# Results

This folder contains the outputs generated from training, evaluation, and demonstration of the two-stage breast cancer analysis system.

The results are intended to support model validation, performance analysis, and interpretability assessment.

---

## Contents

Typical contents of this folder may include:
- Training and validation performance metrics
- Accuracy and loss summaries
- Grad-CAM++ visualization outputs
- Attention weight analysis
- Inference examples from the demo interface

---

## Stage-wise Results

### Stage-1 Screening
- Screening accuracy and validation metrics
- Model behavior analysis for Normal vs Abnormal classification
- Performance evaluation focused on sensitivity

### Stage-2 Multimodal Diagnosis
- Diagnostic accuracy for Benign vs Malignant classification
- Evaluation of multimodal fusion performance
- Analysis of modality attention weights

---

## Interpretability Outputs
- Grad-CAM++ heatmaps for mammogram images
- Grad-CAM++ heatmaps for ultrasound images
- Visual evidence of spatial regions influencing model predictions
